## How it works
- Refresh page to start new round.
- Click HIT :
    - new hand draws for both players
    - value of player hands update
    - algoryhtm checks total ; 
        - if < 21 : draw new card or stand 
        - if > 21 : BUST innerHTML appears (Game Over)
